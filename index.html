🔌 连接 Supabase 数据库

第一步：去 supabase.com 免费注册，新建一个项目
第二步：在项目的 SQL Editor 里运行下面这段建表 SQL：

create table dishes (id bigint generated always as identity primary key, name text, emoji text, cat text, ings text);

create table selected (dish_id bigint primary key, nb boolean default false, yh boolean default false);

create table extras (id bigint generated always as identity primary key, text text, done boolean default false, created_at timestamptz default now());

然后在 Table Editor → 每张表 → RLS 里关闭 Row Level Security（或者添加 allow all policy）
